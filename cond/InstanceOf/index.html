InstanceOf(
  • class: Class,
)

This is a simple instanceof check. Returns the original value if it matches.

import InstanceOf from 'pattahn/cond/InstanceOf';
const VALUE = [1, 2, 3];
expect(InstanceOf(Array).exec(VALUE)).toEqual([VALUE]);
expect(InstanceOf(Object).exec(VALUE)).toEqual([VALUE]);
expect(InstanceOf(Function).exec(VALUE)).toBe(null);
