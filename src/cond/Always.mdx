# Always

The `Always` condition ignores the input value and always returns the values passed to it.

```js
import match from 'matchany';
import Always from 'matchany/cond/Always';

const VALUE = 1;

const result = Always(1, 2, 3).exec('anything');

expect(result).toEqual([1, 2, 3]);
```
