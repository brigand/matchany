# Eq (condition)

Simple equality check. Uses [`Object.is`] when available.

When it matches, it returns the input values.

[`object.is`]:
  https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/is#Description

```js
import Eq from 'pattahn/cond/Eq';

expect(Eq(7).exec(7)).toEqual([7]);
```
