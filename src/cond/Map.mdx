<CondType>Map(transform: values => null | Array)</CondType>

Transform a value with a function, optionally producing new values in the from of an
array.

```js
import match from 'pattahn';
import { Match } from 'pattahn/core';
import Map from 'pattahn/cond/Map';

const VALUE = { x: 7 };

// Take .x, double it, return that.
const xNumber = Map((value) => (typeof value.x === 'number' ? [value.x * 2] : null));

const result = match()
  .with(xNumber, (num) => `match: ${num}`)
  .exec(VALUE);

expect(result).toBe('match: 14');
```
