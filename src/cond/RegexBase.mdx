# RegexBase (helper)

The RegexBase is a helper used internally, but defines the API for `RegexTest`,
`RegexMatch`, and `RegexMatchAll`.

Each of these types can be created in a few different ways. We'll use `RegexTest` for
the examples.

```js
import RegexTest from 'pattahn/cond/RegexTest';
```

You can of course pass a RegExp instance.

```js
expect(RegexTest(/^\d+$/).exec('7')).toEqual(['7']);
expect(RegexTest(new RegExp('^\\d+$')).exec('7')).toEqual(['7']);
```

You can alternatively pass a string, which will be passed to `new Regexp`.

```js
expect(RegexTest('^\\d+$').exec('7')).toEqual(['7']);
```

Or finally, use the factory as a tagged template literal.

```js
expect(RegexTest`^\d+$`.exec('7')).toEqual(['7']);
```
